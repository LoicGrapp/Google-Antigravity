# Google-Antigravity
Voici un repo sur ce que fait, a quoi sert et comment s'utilise Google Antigravity
## Google Antigravity est un IDE classic
Il propose l'auto complention, ainsi qu'un agent ayant le contexte direct de notre projet.
## Google antigravity et son agent intégré
L'agent intégré dans google antigravity nous permet de décrire simplement la fonctionnalité attendue, et l'ia nous ecrira le code, le planifiera, et le verifiera par lui meme
## Vues d'Abstraction Élevées
Il fournit un aperçu clair du travail de l'agent (artefacts, résultats de vérification) pour permettre au développeur d'avoir confiance dans le code produit.
## Cross surface agent
Le Cross-surface Agent d'Antigravity permet à l'IA de contrôler simultanément et de manière coordonnée votre éditeur de code, votre terminal et votre navigateur, afin d'automatiser des flux de travail de développement complexes (planification, codage, test, débogage) à partir d'une simple instruction de haut niveau.
